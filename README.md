# AI Glossary

**AI buzzwords explained. No developer required.**

An interactive glossary of the most important AI terms — explained in plain language for anyone who wants to join the conversation without a technical background.

---

## About

> "I want to create something you could put on a table at a bar — so non-technical people immediately get a feel for these topics and can join the discussion."

That's exactly what this is. AI terms come up constantly in meetings, team chats, and product decisions.
This glossary makes them accessible to everyone, no prior knowledge required.

---

## Content

Terms are grouped into four categories: Prompting, Models & Reasoning, Agents & Skills, and Fundamentals.

---

## Features

- Filterable by category
- Each card expands with a more detailed explanation
- No framework, no dependencies — plain HTML, CSS & JavaScript
- Automatic dark mode via system preference
- Mobile-friendly

---

## Usage

Just open `index.html` in your browser. No build step, no server needed.
